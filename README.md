# Drone Operations Coordinator AI

This is a simple Streamlit-based prototype for a Drone Operations Coordinator AI.

## Current Features
- Upload pilot, drone, and mission CSV files
- View uploaded data in an interactive UI

## Planned Enhancements
- Agentic decision logic (Observe → Reason → Decide → Explain)
- Pilot and drone assignment
- Conflict detection
- Google Sheets integration
- Hosted deployment using Streamlit Cloud

## Tech Stack
- Python
- Streamlit
- Pandas

## How to Run (Locally or via Codespaces)
```bash
pip install -r requirements.txt
streamlit run app.py
